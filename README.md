Видеоплеер

Горячие клавиши

| Клавиша | Действие |
|:-------:|----------|
| `Пробел` / `K` | Play / Pause |
| `→` / `L` | Вперёд 5 сек |
| `←` / `J` | Назад 5 сек |
| `Ctrl + →` | Вперёд 30 сек |
| `Ctrl + ←` | Назад 30 сек |
| `↑` | Громкость + |
| `↓` | Громкость - |
| `M` | Выкл/Вкл звук |
| `F` / `F11` | Полный экран |
| `Esc` | Выйти из полноэкранного |
| `N` | Следующее видео |
| `P` | Предыдущее видео |
| `[` | Скорость - |
| `]` | Скорость + |
| `Backspace` | Сброс скорости (1x) |
| `Home` | В начало |
| `End` | В конец |
| `Ctrl + O` | Открыть файл |
| `Ctrl + Q` | Выход |
| `F5` | Перезагрузить |

---

**Запуск:** `python main.py`
